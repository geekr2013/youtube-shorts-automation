# YouTube Shorts 자동화

매일 자동으로 Pexels에서 동영상을 수집하고, Gemini AI로 한글 제목/스크립트를 생성하여 YouTube Shorts로 업로드합니다.

## 🎯 기능

- ✅ Pexels API로 로열티 프리 동영상 수집
- ✅ Gemini AI로 한글 제목/스크립트 자동 생성
- ✅ gTTS로 한국어 나레이션 추가
- ✅ YouTube Shorts 자동 업로드
- ✅ 결과 이메일 자동 발송
- ✅ GitHub Actions로 매일 자동 실행

## 🚀 사용 방법

1. 이 저장소를 Fork
2. GitHub Secrets 설정
3. 매일 자동 실행됨!

## 🔑 필요한 API 키

- Pexels API Key
- Gemini API Key
- Gmail 앱 비밀번호
- YouTube OAuth 인증

## 📅 실행 일정

- 매일 오전 9시 (한국 시간)
- GitHub Actions로 자동 실행
